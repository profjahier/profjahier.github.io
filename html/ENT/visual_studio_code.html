Visual Studio Code

Présentation rapide

Visual Studio Code est un éditeur de code que j'aime bien ! Son autocomplétion est très pratique et efficace. En plus, il intègre git + github.

Installation

Ce logiciel est multiplatforme et gratuit. Il n'est pas libre en revanche (dommage).

Il peut être installé de façon autonome, mais sachez que la distribution Python Anaconda propose son installation automatique en complément (je recommande cette façon de faire si vous devez installer Python sur une machine).

Le site officiel propose évidemment des liens de téléchargement pour chaque configuration de machine.

Personnalisation et compléments

L'interface utilisateur est évidemment largement paramétrable en fonction de vos goût.

De nombreuses extensions très utiles peuvent être ajoutées à la version de base, par exemple pour travailler avec Python ou Javascript.

Indentation automatique

Une très bonne fonctionnalité pour remettre un peu d'ordre dans un code mal indenté !

Ctrl + Maj + I

Cela remet tout d'équerre automatiquement, et votre code redevient lisible.

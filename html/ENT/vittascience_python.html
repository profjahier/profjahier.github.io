Programmer Python avec l'interface Vittascience

Vous pouvez programmer librement avec l'interface ci-dessous :

Il est possible de programmer avec des Blocs de code visuels, ou en mode Hybride ou en mode Code pur : ces modes sont accessibles avec les icônes de l'interface en haut à droite.
Ensuite on peut exécuter le code en cliquant sur le bouton, et on accède alors à la console interactive REPL en-dessous.

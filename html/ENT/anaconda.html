Anaconda - Python

Présentation rapide

Anaconda est une distribution Python qui inclut par défaut de nombreux modules complémentaires scientifiques comme Matplotlib, ou Numpy par exemple.

La distribution installe aussi un serveur Jupyter pour éditer et exécuter des notebooks, ainsi que l'IDE (Integrated Develpoment Environment) Spyder, et propose l'installation complémentaire de Visual Studio Code (non libre) pour lequel j'ai beaucoup d'affinité.

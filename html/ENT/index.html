Outils numériques, Environnement de travail

Avertissement

Enseignant de physique/chimie et numérique/informatique au lycée, je développe ce site principalement à l'intention de mes élèves. Ce site hebergé par Github contient 3 sections principales : Physique/Chimie, NSI et Environnement Numérique de Travail (ici-même).

Dans cette section du site, je présenterai différents outils qui me semblent utiles pour développer un environnement de travail numérique agréable et efficace.

Bien entendu, mes choix de logiciels ne sont pas forcément les seuls ou les meilleurs du marché (bien souvent, un choix de logiciel pour une tache donnée est assez subjectif) mais ils sont au moins satisfaisants pour une utilisation efficace pour un lycéen ou un professeur du 2nd degré. J'essaie d'utiliser au maximum des logiciels libres, multiplatformes et gratuits.

Ce site, encore très largement en construction, présente néanmoins quelques pages qui peuvent déjà être consultées avec intérêt (en toute objectivité et modestie, bien entendu !).

Au menu pour l'instant...

  • Anaconda et Python : distribution scientifique de Python très complète. Idéale pour avoir une bonne installation de Python (+ serveur Jupyter local).
  • Git et Github : gestion de versions.
  • Visual Studio Code : éditeur de code.
  • Présenter du code en ligne : exemples de présentation de code avec Prism.
  • MathJax : permet d'inclure des formules mathématiques (Latex, MathML) dans les pages Web.
  • Python en ligne : Intégration de la plateforme Vittascience permettant de coder dans un navigateur sans installation, en mode Blocs ou en mode Code.

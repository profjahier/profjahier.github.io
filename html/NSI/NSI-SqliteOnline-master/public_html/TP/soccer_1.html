Quelques statistiques sur le championnat 2015-2016

Vérifiez qu'il y avait 20 équipes à participer au championnat. 
SELECT COUNT(*) FROM Team;
Déterminez la date de création la plus ancienne de ces clubs. 
SELECT MIN(birthday) FROM Team;
Déterminez le nom du ou des clubs créés à cette date. 
SELECT name FROM Team WHERE birthday=1881;
Affichez dans l'ordre de leur création, le plus ancien d'abord, le nom et la date de création des clubs ayant disputé ce championnat. Vous utiliserez un ORDER BY. 
SELECT name, birthday FROM Team ORDER BY birthday;
Vérifiez qu'il y a bien eu 380 matchs disputés. 
SELECT COUNT(*) FROM Match;
Vérifiez qu'il y a bien eu 66 penaltys marqués. 
SELECT COUNT(*) FROM Event WHERE type='penalty';
Et que 78 cartons rouges ont été distribués. 
SELECT COUNT(*) FROM Event WHERE type='red';
Vérifiez qu'il y a bien eu 93 penaltys sifflés. Vous utiliserez un WHERE OR. 
SELECT COUNT(*) FROM Event WHERE type='penalty' OR type='miss';
Vérifiez qu'il y a bien eu 960 buts. Vous utiliserez un WHERE IN. 
SELECT COUNT(*) FROM Event WHERE type IN ('penalty','goal','own');
Affichez les 10 cartons rouges obtenus le plus rapidement durant un match. Vous utiliserez un ORDER BY et un LIMIT. Vous noterez bien l'identifiant du joueur et celui de l'équipe asociée au carton rouge le plus rapide. 
SELECT * FROM Event WHERE type='red' ORDER BY time LIMIT 10;

La relation Agences

Au départ ce réseau se limitait à la région Bretagne mais peu à peu vous avez ouvert de nombreuses agences et maintenant le réseau couvre tout le territoire national. Un certain nombre d'informations relatives à ces agences sont stockées dans la base de données. Par exemple, vous pouvez obtenir les noms des enseignes ou les villes dans lesquelles elles sont situées. Plus précisément, la relation Agences est décrite par :



idINTla clef primaire
nomTEXTvous avez de l'humour
adresseTEXTrue, avenue, boulevard...
codeTEXTle code postal
villeTEXTla ville
paysTEXTle pays
Visualisez toute la relation Agences. 
SELECT * FROM Agences
Listez uniquement les noms des agences et leur ville. 
SELECT nom, ville FROM Agences
Listez les noms des agences de la ville de Lorient. 
SELECT nom FROM Agences WHERE ville='Lorient'
Listez les noms des agences du département du Morbihan (code postal 56***) ainsi que les codes postaux en utilisant un WHERE LIKE. 
SELECT nom, code FROM Agences WHERE code LIKE '56%'
Listez les noms des agences du département du Morbihan ainsi que les codes postaux en utilisant la fonction SUBSTR(str,first,nb) dans un WHERE. 
SELECT nom, code FROM Agences WHERE SUBSTR(code,1,2) = '56'
Listez les villes du département du Morbihan hébergeant au moins une agence. Attention, une ville ne doit apparaître qu'une seule fois. 
SELECT DISTINCT ville FROM Agences WHERE code LIKE '56%'
Listez les noms des agences en Bretagne ('56','29','35','22') ainsi que les codes postaux en utilisant la fonction SUBSTR(str,first,nb) et la directive IN (element1,element2,element3,...) 
SELECT nom, code FROM Agences WHERE SUBSTR(code,1,2) IN ('56','29','35','22')
Déterminez le nombre d'agences en Bretagne. 
SELECT COUNT(*) FROM Agences WHERE SUBSTR(code,1,2) IN ('56','29','35','22')
Déterminez le nombre d'agences en France. 
SELECT COUNT(*) FROM Agences WHERE pays='France'

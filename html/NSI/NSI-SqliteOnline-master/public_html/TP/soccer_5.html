Un peu de fiction

Vous avez les moyens de modifier l'histoire et vous décidez de ne pas vous en priver. Vous allez commencer par réécrire le scénario du derby breton de septembre dont l'identifiant est 129 dans la relation Match.

Affichez la liste des évènements associés à ce match. 
SELECT * FROM Event where match=129;
Ajoutez le nom du joueur concerné. 
SELECT Event.*, Player.name FROM Event 
JOIN Player ON Event.player=Player.id
WHERE match=129;
Ajoutez le nom de l'équipe concernée. 
SELECT Event.*, Player.name, Team.city FROM Event 
JOIN Player ON Event.player=Player.id
JOIN Team ON Event.team=Team.id
WHERE match=129;
Limitez l'affichage à l'identifiant et à la minute de l'évènement, au nom du joueur, à celui de l'équipe et au type de l'évènement. L'affichage doit s'effectuer dans l'ordre chronologique. 
SELECT Event.id, Event.time, Player.name, Team.city, Event.type FROM Event 
JOIN Player ON Event.player=Player.id
JOIN Team ON Event.team=Team.id
WHERE match=129
ORDER BY time;
Vous vous souvenez que le but de Dembele n'était pas valide. Vous décidez de transformer ce but en carton rouge. Il vous faut utiliser UPDATE avec l'identifiant de l'évènement. 
UPDATE Event SET type='red' WHERE id=301
Par contre le but refusé de Abdul Majeed Waris (identifiant 149) pour Lorient (identifiant 18) à la soixantième minute était parfaitement correct. Mais avant de l'ajouter il faut déterminer un identifiant unique pour la table Event. Récupérez donc la valeur maximale des identifiants de cette table. 
SELECT Max(id) from Event;
Ajoutez donc ce superbe but avec une requête INSERT INTO 
INSERT INTO Event(id,match,time,player,team,type)
VALUES(2600,129,60,149,18,'goal');
Et puis le but de Jeremie Boga n'était pas non plus valable. Supprimez donc l'évènement d'identifiant 302. 
DELETE FROM Event WHERE id=302;
Il ne reste plus qu'à visualisez à nouveau ce scénario. 
SELECT Event.time, Player.name, Team.city, Event.type FROM Event 
JOIN Player ON Event.player=Player.id
JOIN Team ON Event.team=Team.id
WHERE match=129
ORDER BY time;
